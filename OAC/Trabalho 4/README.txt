EXECUTANDO:

1. Abra um editor de códigos de sua preferência (sugestão da equipe desenvolvedora: Visual Studio Code) ou o terminal na pasta que contém todos os arquivos;

*OBS: Todos os arquivos devem estar na mesma pasta (main.py, registers.py, memory.py, if/id/ex/mem/wb_stage.py);

2. Digite no terminal "python main.py"

3. O programa estará rodando e poderá ser visto cada ciclo de clock apertando a tecla [ENTER];

4. Quando forem executados todos os ciclos de clock necessários, o programa será encerrado sozinho.

---------------------------------------------------------------------------------------------------------------------------------------
TESTANDO COM SEU ASSEMBLY:
** Deseja colocar seu código ASSEMBLY para testar? Siga os passos:

1. No editor de código, abra o arquivo "main.py";

2. Digite em "program" o código ASSEMBLY que deseja testar;

*IMPORTANTE! OBS: Cada linha de instrução deverá ser uma string sem espaços entre os registradores e separadas por vírgula ao final, observe:
	program = [
        	"addi x1,x0,5",  # os caractéres "" englobam a instrução e não há espaço entre x1,x0,5, e no final da linha há uma virgula. 
        	"addi x2,x0,10"
	]

3. Salve o código alterado;

4. execute como no passo 2 da seção "EXECUTANDO".
